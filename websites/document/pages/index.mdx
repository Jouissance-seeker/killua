# Introduction

## What is killua?

Killua is a local-storage management library for React applications.

## Features

- Get data from local storage
- Set data to local storage
- Reducer for state management
- Selector for data access
- Expiration timer
- Schema validation
- Obfuscate data in local storage
- Server-Side Compatibility
- TypeScript friendly
- Auto update in other tabs
- Auto update in other components
- Config file for configuration management

## Installation

The Killua package lives in [npm](https://www.npmjs.com/package/killua). To install the latest stable version, run the following command:

```bash
# npm
npm install killua

# pnpm
pnpm install killua

# yarn
yarn add killua
```
